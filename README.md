## 🎯 What is Sumr?

**Sumr** is an AI-powered **YouTube Video Summarization SaaS Platform** that transforms long videos into concise, readable summaries. Save hours watching content - get the key insights in seconds.

---

## ✨ Features

- 📺 **YouTube Video Processing** - Paste any YouTube URL
- 🤖 **AI-Powered Summarization** - Smart content extraction
- 📝 **Multiple Export Formats** - PDF, text, copy to clipboard
- 🎨 **3D Interactive UI** - Three.js powered animations
- ⚡ **Fast Processing** - Get summaries in seconds
- 📱 **Mobile Responsive** - Works on all devices
- 🌙 **Dark Mode** - Easy on the eyes

## 🚀 Quick Start

### Prerequisites

- Node.js 18+ installed
- npm or yarn package manager

### Installation

```bash
# Clone the repository
git clone https://github.com/raghavshahhh/sumr.git
cd sumr

# Install dependencies
npm install

# Run development server
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) in your browser.

## 📜 Available Scripts

| Command | Description |
|---------|-------------|
| `npm run dev` | Start development server |
| `npm run build` | Build for production |
| `npm start` | Start production server |

## 📊 How It Works

1. **Paste YouTube URL** - Enter any video link
2. **Extract Transcript** - Automatic transcript fetching
3. **AI Summarization** - Generate concise summary
4. **Export Results** - Download as PDF or copy text
